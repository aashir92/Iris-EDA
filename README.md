# Iris-EDA

## Task Objective
Perform exploratory data analysis on the Iris dataset to identify distributions, species-wise trends, and outlier patterns.

## Dataset Used
- Iris dataset loaded via `seaborn.load_dataset("iris")`
- Features: `sepal_length`, `sepal_width`, `petal_length`, `petal_width`
- Label: `species`

## Models Applied
No predictive model is applied in this task. The work is focused on descriptive analysis and visualization.

## Key Results and Findings
- Petal features provide stronger class separation than sepal features.
- Setosa is clearly separable from versicolor and virginica.
- Versicolor and virginica overlap in some feature regions.
- Outlier checks show a few extreme points but stable overall class structure.
